Marketing Funnel & Conversion Performance Analysis

This project focuses on analyzing marketing campaign performance using the Bank Marketing Dataset in Power BI. The dashboard identifies customer conversion trends, funnel drop-offs, campaign effectiveness, and high-performing customer segments to support data-driven marketing decisions.

📌 Objective

The objective of this project is to:

Analyze marketing funnel performance
Identify major conversion drop-offs
Evaluate campaign effectiveness
Discover high-converting customer segments
Generate actionable business insights

🛠 Tools & Technologies Used
Power BI
Power Query
DAX
Data Visualization
Bank Marketing Dataset

📊 Dashboard Features
*KPI Cards
  Total Contacts
  Converted Customers
  Conversion Rate
  Average Call Duration
*Funnel Analysis
  Total Contacts
  Interested Leads
  Qualified Leads
  Converted Customers
*Conversion Trend Analysis
*Contact Method Performance
*Customer Segmentation
  *Age Group Analysis
  *Job-wise Conversion Analysis
*Interactive Filters & Slicers

💡 Key Insights
Major customer drop-off occurs during the lead qualification stage
Cellular communication generated the highest conversions
Customers aged 26–45 showed stronger conversion behavior
Excessive campaign contacts negatively impacted conversion rates
Management and technician professions contributed more conversions

🚀 Key Takeaways
Improve lead qualification strategies to reduce funnel drop-offs
Focus marketing campaigns on high-converting customer segments
Prioritize cellular communication for better campaign performance
Reduce unnecessary repeated customer contacts
Use data-driven targeting to improve overall conversion efficiency
